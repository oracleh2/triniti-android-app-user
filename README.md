# Triniti User - Android App Releases

Репозиторий для релизов пользовательской версии Android приложения Triniti.

## Установка

Скачайте последний APK файл из раздела [Releases](https://github.com/oracleh2/triniti-android-app/releases).

## Версии

Все версии приложения доступны в разделе Releases с тегами формата `v1.0.0`.

## Требования

- Android 8.0 (API 26) или выше
- Разрешение на установку из неизвестных источников

## Changelog

История изменений доступна в описании каждого релиза.
